# Multidisciplinary Project – Smart AI and IoT System

This is a university project at Ho Chi Minh City University of Technology (HCMUT), combining AI and IoT with YoloBit V8.

> 🚧 Project is still in development.

## Features

- 🔥 Fire detection using a YOLOv5 model
- 👤 Face recognition (coming soon)
- 🌐 Web interface for monitoring and control (in progress)
- 📡 Communication between AI module and YoloBit using UART
- 🔔 Real-time alerts and automation

## Files

- `main.py`: Main program
- `fire.py`: Fire detection logic
- `uart.py`: UART communication
- `best.pt`: YOLOv5 trained model
- `fire_detected.jpg`: Sample detection image

## Tech Used

- Python, OpenCV, PyTorch
- YoloBit V8, MicroPython
- Flask (planned for web)
- YOLOv5 (fire detection)
